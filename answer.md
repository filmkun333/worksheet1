# แบบฝึกหัด Git

ชื่อ-นามสกุล : ธีรวัฒน์ มะโนวร

ห้อง : 2/2

เลขที่ : 7

----------------------------------

ข้อที่ 1
Git คืออะไร

ตอบ : ระบบควบคุมเวอร์ชัน (Version Control System)

----------------------------------

ข้อที่ 2
Repository คืออะไร

ตอบ : พื้นที่จัดเก็บไฟล์โปรเจ็กต์และประวัติการเปลี่ยนแปลงทั้งหมด โดยทำงานผ่านระบบ Git

----------------------------------

ข้อที่ 3
คำสั่งใดใช้ Clone Repository

ตอบ : git clone โดยเป็นการคัดลอกไฟล์และประวัติทั้งหมดจากเซิร์ฟเวอร์ (Remote Repository) เช่น GitHub ลงมาไว้ในเครื่องคอมพิวเตอร์

----------------------------------

ข้อที่ 4
คำสั่งใดใช้เพิ่มไฟล์เข้าสู่ Staging Area

ตอบ : git add

----------------------------------

ข้อที่ 5
คำสั่งใดใช้บันทึกการเปลี่ยนแปลง

ตอบ : git commit

----------------------------------

ข้อที่ 6
คำสั่งใดใช้ส่งข้อมูลขึ้น GitHub

ตอบ : git push

----------------------------------

ข้อที่ 7
วันนี้ได้เรียนรู้อะไรเกี่ยวกับ Git

ตอบ : git clone

----------------------------------
